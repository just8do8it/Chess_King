Solving the 3x3x3 Rubik's cube with one hand

Once you learn the CFOP speedcubing method of solving the cube and get decent at it, you can think of a harder task to complete - solving the cube with only one hand. This might be pretty difficult at first, but you actually need to change only some of the algorithms for the last layer, which isn't impossible after all.
